# Mask Possession Game

## How the repo is struct:
    - Folder struct:
        - **ASSETS** contains every asset file (ie. .blend files of furniture, .png files of ambiental or VFX...)
        Must be subdivided in subfolder named by category (/furniture, /VFX, /room, /texture...)
        - **AUDIO** contains all the audio file subdivided in /SFX and /Music
        - **CHARACTER** contains all the assets and scenes of the characters subdivided by folders (/PLAYER is the most important) a subsubfolder with textures and player related script is welcomed
        - **SCRIPTS** contains the essential scripts for components and classes
        - **SCENES** contains all the scenes divided in rooms, levels... (Furniture, character and similar scenes must be putted in their own folder)

    - Any file of any extension whose name start with _'PRSL_'_ would be ignored
    - If you are stilll working on a file plz name it with the suffix _WIP (ie. super_cat_WIP.png)
    - plz do not use any spaces in folder or file name, use snake_case instead and reserve ALL CAPS for important information and metadata

## To push changes
To push changes plz make sure that everything is in the right folder, if you push changes from browser, plz send just the thing you did work on respecting the folder structure.  (ie. if you worked on stitch_the_bitch.png and it is a secondary character import the folder CHARACTER/stitch_the_bitch/stitch_the_bitch.png)

In the name of the commit plz copy/paste the task as shown in the TASK.md file. in the description plz say what you worked on, indicate the name of the file you worked on and add a small description of your job.

If you are not sure put it on the discord or write on wa or whatever.
